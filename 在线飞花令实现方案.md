# 在线飞花令功能实现方案

## 功能概述

实现在线飞花令对战功能，包括：
1. 单人飞花令和在线飞花令两种模式
2. 在线用户列表展示（姓名、班级、最高记录）
3. 邀请对战功能
4. 实时对战逻辑
5. 扔题特权（每人3次）
6. 最高记录统计

## 已完成的工作

### 1. 数据库表设计 ✅

#### feihua_battles 表（对战记录）
```sql
CREATE TABLE feihua_battles (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  player1_id INTEGER NOT NULL,
  player2_id INTEGER NOT NULL,
  keyword TEXT NOT NULL,
  winner_id INTEGER,
  loser_id INTEGER,
  total_rounds INTEGER DEFAULT 0,
  player1_throw_count INTEGER DEFAULT 0,
  player2_throw_count INTEGER DEFAULT 0,
  battle_history TEXT,
  started_at TEXT NOT NULL,
  ended_at TEXT,
  FOREIGN KEY (player1_id) REFERENCES users(id),
  FOREIGN KEY (player2_id) REFERENCES users(id)
)
```

#### feihua_high_records 表（最高记录）
```sql
CREATE TABLE feihua_high_records (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  keyword TEXT NOT NULL,
  max_rounds INTEGER DEFAULT 0,
  total_battles INTEGER DEFAULT 0,
  wins INTEGER DEFAULT 0,
  losses INTEGER DEFAULT 0,
  updated_at TEXT NOT NULL,
  FOREIGN KEY (user_id) REFERENCES users(id),
  UNIQUE(user_id, keyword)
)
```

### 2. WebSocket连接服务 (feihualingSocket.js)

```javascript
import io from 'socket.io-client'

class FeiHuaLingSocket {
  constructor() {
    this.socket = null
    this.listeners = new Map()
  }
  
  connect(token) {
    this.socket = io('http://localhost:3000', {
      auth: { token }
    })
    
    this.setupListeners()
  }
  
  setupListeners() {
    // 在线用户列表更新
    this.socket.on('online-list-update', (users) => {
      this.emit('online-users', users)
    })
    
    // 接收邀请
    this.socket.on('receive-invitation', (invitation) => {
      this.emit('receive-invitation', invitation)
    })
    
    // 游戏开始
    this.socket.on('game-start', (gameData) => {
      this.emit('game-start', gameData)
    })
    
    // 诗句提交成功
    this.socket.on('poem-submitted', (data) => {
      this.emit('poem-submitted', data)
    })
    
    // 扔题成功
    this.socket.on('question-thrown', (data) => {
      this.emit('question-thrown', data)
    })
    
    // 游戏结果
    this.socket.on('game-result', (result) => {
      this.emit('game-result', result)
    })
    
    // 对手断线
    this.socket.on('opponent-disconnected', (data) => {
      this.emit('opponent-disconnected', data)
    })
    
    // 错误处理
    this.socket.on('error', (error) => {
      this.emit('error', error)
    })
  }
  
  // 发送邀请
  sendInvitation(targetUserId, keyword, difficulty) {
    this.socket.emit('send-invitation', {
      targetUserId,
      keyword,
      difficulty
    })
  }
  
  // 接受邀请
  acceptInvitation(inviteId, inviterId) {
    this.socket.emit('accept-invitation', {
      inviteId,
      inviterId
    })
  }
  
  // 拒绝邀请
  rejectInvitation(inviteId, inviterId) {
    this.socket.emit('reject-invitation', {
      inviteId,
      inviterId
    })
  }
  
  // 提交诗句
  submitPoem(roomId, poem) {
    this.socket.emit('submit-poem', {
      roomId,
      poem
    })
  }
  
  // 扔题
  throwQuestion(roomId) {
    this.socket.emit('throw-question', {
      roomId
    })
  }
  
  // 超时
  timeout(roomId) {
    this.socket.emit('timeout', {
      roomId
    })
  }
}

export default new FeiHuaLingSocket()
```

## 游戏流程

### 1. 进入在线模式
1. 用户选择"在线对战"模式
2. 连接WebSocket并认证
3. 显示在线用户列表

### 2. 发起邀请
1. 点击在线用户的"邀请"按钮
2. 选择令字和难度
3. 发送邀请给目标用户

### 3. 接受邀请
1. 被邀请用户收到弹窗
2. 显示邀请者信息（姓名、班级、最高记录）
3. 选择接受或拒绝

### 4. 游戏进行
1. 双方进入游戏房间
2. 显示令字、当前回合、剩余时间
3. 轮流回答诗句
4. 可以使用扔题特权（每人3次）
5. 超时判负

### 5. 游戏结束
1. 一方超时或答不上来
2. 显示游戏结果
3. 保存对战记录
4. 更新最高记录
5. 返回在线用户列表

## 特殊功能

### 扔题特权
- 每人每局游戏有3次扔题机会
- 扔题后，对方必须回答当前题目
- 扔题次数用完后按钮禁用

### 超时处理
- 根据难度设置不同时间限制
  - 简单：60秒
  - 中等：45秒
  - 困难：30秒
- 超时自动判负

### 断线处理
- 用户断线自动判负
- 保存对战记录
- 通知对方获胜

## 数据统计

### 最高记录
- 按令字统计最高轮数
- 记录总对战次数
- 记录胜负场次

### 对战历史
- 保存每场对战详情
- 记录使用的诗句
- 记录扔题次数
