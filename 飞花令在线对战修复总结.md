## 🧪 测试建议

### 基础功能测试
1. **用户登录测试**
   - 打开飞花令在线对战页面
   - 检查是否能成功连接Socket
   - 检查是否能获取在线用户列表

2. **邀请功能测试**
   - 选择难度和令字
   - 查看在线用户列表
   - 点击邀请按钮
   - 检查被邀请者是否收到邀请弹窗

3. **游戏开始测试**
   - 接受邀请
   - 检查游戏是否正常开始
   - 检查倒计时是否正常

4. **诗句提交测试**
   - 输入包含令字的诗句
   - 提交诗句
   - 检查诗句是否被验证
   - 检查轮次是否正确切换

5. **扔题功能测试**
   - 点击扔题按钮
   - 检查扔题次数是否正确
   - 检查轮次是否切换

6. **游戏结束测试**
   - 一方回答错误或超时
   - 检查游戏是否正确结束
   - 检查结果是否正确显示

### 异常情况测试
1. **断线测试**
   - 游戏中关闭浏览器
   - 检查对手是否收到断线提示
   - 检查游戏是否正确结束

2. **重连测试**
   - 断线后30秒内重新连接
   - 检查是否能恢复游戏

3. **并发测试**
   - 同时邀请多个用户
   - 检查邀请是否正常发送

## 📝 使用说明

### 启动服务
```bash
# 启动后端
cd backend
node server.js

# 启动前端（新终端）
cd frontend
npm run dev
```

### 测试流程
1. 打开两个浏览器窗口（或使用两个不同的浏览器）
2. 分别登录两个不同的用户账号
3. 在两个窗口中都打开飞花令在线对战页面
4. 用户A选择难度和令字
5. 用户A查看在线用户列表，应该能看到用户B
6. 用户A点击邀请用户B
7. 用户B应该收到邀请弹窗
8. 用户B接受邀请
9. 游戏开始，双方轮流回答诗句
10. 测试扔题功能
11. 测试游戏结束
