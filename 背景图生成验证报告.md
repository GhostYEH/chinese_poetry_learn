## 🎯 功能流程

### 完整工作流程
```
用户打开诗词详情页
    ↓
前端调用 /api/ai/image/pregenerate
    ↓
后端调用硅基流动API生成图片
    ↓
API返回图片URL
    ↓
通过Socket.io推送图片URL到前端
    ↓
前端缓存图片URL
    ↓
用户点击"开始学习"按钮
    ↓
显示AI生成的背景图
```

### Socket事件流
```
前端监听：
  - image-generate-pending  → 显示"诗中有画，画境渐生..."
  - image-generate-success → 设置背景图URL
  - image-generate-fail    → 显示错误提示，使用默认背景

后端发送：
  - image-generate-pending → 图片生成开始
  - image-generate-success → 图片生成成功，包含URL
  - image-generate-fail    → 图片生成失败，包含错误信息
```

## 🔧 技术细节

### API调用参数
```javascript
{
  model: 'Kwai-Kolors/Kolors',
  prompt: '根据古诗词《标题》（作者：作者）的内容生成一幅中国风图像...',
  image_size: '1440x720',
  batch_size: 1,
  num_inference_steps: 20,
  guidance_scale: 7.5
}
```

### Prompt模板
```
根据古诗词《{title}》（作者：{author}）的内容生成一幅中国风图像，
画面要准确描绘诗中所描述的场景和意境，
{content}，
中国传统风格，高清细腻，氛围感强，
无任何文字、无水印、无logo，
画面干净统一，适合做网页背景图
```

### 缓存机制
- **缓存时长**：30天
- **缓存键**：`{poemId}_{title}_{author}` 的MD5哈希
- **缓存存储**：内存Map对象
- **缓存命中**：相同诗词复用缓存图片

## ⚠️ 注意事项

### 1. API调用限制
- 硅基流动API有调用频率限制
- 当前设置：每分钟最多5次请求
- 如遇429错误，请等待后重试

### 2. 图片生成时间
- 首次生成需要10-30秒
- 建议在页面加载时预生成
- 用户点击时图片已准备好

### 3. 网络要求
- 需要访问 `api.siliconflow.cn`
- 需要访问 `s3.siliconflow.cn`（图片存储）
- 确保网络连接正常

### 4. 错误处理
- API密钥无效：显示错误提示，使用默认背景
- 网络错误：显示错误提示，使用默认背景
- API限流：显示错误提示，使用默认背景
- 任何错误都不会影响诗词学习功能
